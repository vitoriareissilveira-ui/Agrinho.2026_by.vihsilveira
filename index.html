Fazenda Fortaleza (Invernada da Ventania)

📜 História: Uma das propriedades mais antigas, fundada no século XIX. Foi aqui que ocorreu o evento que deu nome à cidade: em 1870, um forte vento derrubou a mata, tornando a terra própria para o cultivo 🪓🌪️

📍 Localização: Região histórica central, zona rural de Ventania - PR

📦 Produção: Milho, feijão, criação de gado de corte e leiteiro 🐄🌽🥛

♻️ Meio Ambiente: Preserva áreas de Mata Atlântica, matas ciliares e nascentes importantes para a região 🌿💧

👤 Proprietário histórico: Francisco Pinheiro das Chagas Ventania 📝

Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de apresentar a riqueza da produção rural da nossa cidade, Ventania, localizada nos Campos Gerais do Paraná. Aqui destacamos propriedades históricas, familiares e empresas que unem tradição, trabalho e tecnologia para levar alimentos de qualidade para toda a região. Cada marcador está posicionado exatamente no local onde a propriedade se encontra, com as distâncias reais existentes entre elas.

Utilizamos linguagens de programação como HTML, CSS e JavaScript para estruturar, estilizar e dar funcionalidade ao site, alinhando tecnologia com o meio rural, tema central do Agrinho Tec 2026.
